Automatic Street Light System

An IoT-based smart street lighting system using ESP32, LDR sensor, LEDs, and Telegram Bot for automatic light control and remote monitoring.

📌 Overview

The Automatic Street Light System is designed to reduce electricity wastage by automatically controlling street lights based on surrounding light intensity.

The system uses an LDR sensor to detect day and night conditions. When darkness is detected, the ESP32 turns ON the street lights (LEDs). During daytime, lights are switched OFF automatically.

A Telegram Bot is integrated for real-time monitoring and control.

🚀 Features
Automatic ON/OFF street lights
ESP32 WiFi enabled controller
LDR based light detection
Telegram Bot notifications
Remote control via Telegram
Energy saving system
Low maintenance cost.

⚙️ Working Principle
LDR senses ambient light.
If darkness detected → LEDs ON.
If daylight detected → LEDs OFF.
Telegram bot sends status updates.
